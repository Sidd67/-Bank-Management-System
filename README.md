# Bank Management System

A C/C++ based console application for managing bank accounts supporting deposits, withdrawals, transfers, and account management.

## Tech Stack
* C/C++ - Core programming language
* * File I/O - Data persistence
  * * OOP Concepts - Account class design
   
    * ## Features
    * * Create Account - Register new bank accounts with personal details
      * * Deposit Money - Add funds to any account
        * * Withdraw Money - Withdraw funds with balance validation
          * * View Account - Display account details and balance
            * * Search Account - Find accounts by account number
              * * Update Account - Modify account holder information
                * * Delete Account - Remove accounts from the system
                  * * View All Accounts - List all registered accounts
                    * * File Persistence - Data saved to binary files
                     
                      * ## Project Structure
                      * ```
                        Bank-Management-System/
                        |-- bank.cpp         # Main source file
                        |-- bank.h           # Header file with class definitions
                        |-- records.dat      # Binary data file (auto-generated)
                        \-- README.md
                        ```

                        ## Getting Started

                        Prerequisites: GCC/G++ compiler

                        Compile and run:
                        ```
                        git clone https://github.com/Sidd67/-Bank-Management-System.git
                        cd -Bank-Management-System
                        g++ bank.cpp -o bank
                        ./bank
                        ```
